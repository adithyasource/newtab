newtab.txt

an ultra simple and lightweight product that i’ve built where you can just write shit down in the new tab. works without an internet connection.



check out the project on github

download for

chromium (chrome, edge, arc, opera)
firefox


✦ all your text is stored and stays there even if you close the tab

✦ bold italic and underline functionality

✦ hover to the top left of the screen for the sidebar!

✦ beautiful darkmode with alt + q

✦ quickly blur your screen if someone comes around with ctrl +q

✦ download your text as a .txt file with ctrl + s

✦ change the font to fit your aesthetic
